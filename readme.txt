# Therestella Foundation Website

## Overview

Welcome to the official website of **TheRestella Foundation**, a nonprofit organization dedicated to empowering underprivileged communities through education and sustainable development initiatives.This website serves as a platform to share our mission, ongoing projects, events, and ways to get involved.

## Features

- **Homepage**: Introduction to the foundation, mission statement, and key initiatives.
- **About Us**: Detailed information about the organization, its history, team members, and goals.
- **Projects**: Overview of ongoing and completed projects with detailed descriptions.
- **Events**: Calendar and details of upcoming events and activities.
- **Get Involved**: Information on how to volunteer, donate, or partner with TheRestella Foundation.
- **Contact Us**: Contact form and details for reaching out to the foundation.
- **Responsive Design**: Optimized for viewing on all devices, including mobile, tablet, and desktop.

## Technologies Used

- **HTML5**: For structuring the content of the website.
- **CSS3**: For styling and layout.
- **SCSS**: For more efficient and maintainable CSS code.
- **JavaScript**: For interactive elements and enhancing user experience.
- **PHP**: For backend processing, including form handling and dynamic content rendering.

## Getting Started

### Prerequisites

- A web server or local server environment (like XAMPP, WAMP, or MAMP) for PHP support.
- Basic knowledge of HTML, CSS, JavaScript, and PHP.

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/devkobby24/therestella-foundation-website.git
   cd therestella-foundation-website
   ```

2. If running locally, ensure your server environment (e.g., XAMPP, WAMP) is running.

3. Place the project files in your server's root directory (e.g., `htdocs` for XAMPP).

4. Open your web browser and navigate to `http://localhost/therestella-foundation-website/` to view the website.

### File Structure

- **index.html**: The main homepage of the website.
- **about.html**: The About Us page.
- **projects.html**: The Projects page.
- **events.html**: The Events page.
- **get-involved.html**: The Get Involved page.
- **contact.php**: The Contact Us page with PHP form handling.
- **assets/**: Contains all assets like images, fonts, etc.
- **css/**: Contains the compiled CSS files.
- **scss/**: Contains the SCSS files.
- **js/**: Contains JavaScript files for interactivity.

### Usage

1. **Customize Content**: Update the HTML files to reflect the specific content and branding of TheRestella Foundation.
2. **Style Customization**: Modify the SCSS files under the `scss/` directory to change the look and feel of the website.
3. **Form Handling**: The contact form on the Contact Us page is handled by PHP. Ensure your server environment supports PHP to test this functionality.
